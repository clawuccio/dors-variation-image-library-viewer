# Dors variation image library viewer

Static GitHub-friendly viewer for the extracted Dors variation image library.

## Included
- `index.html` — searchable gallery viewer
- `dors-variation-image-library.json` — structured match dataset
- `dors-variation-image-library.csv` — CSV export
- `images/` — extracted reference images

## Match types
- `direct_grid`
- `direct_single_column`
- `hero_single_code_page`

## Notes
This repo is optimized for easy browsing on GitHub and as a static site. Direct crops are highest-confidence; hero-page fallbacks are clearly labeled.
